Ceci est le code source de la partie mobile d'Aura.

Le code est fonctionnel à ce jour sur Android. L'équipe ne pouvant pas tester sur IOS, nous ne
garantissons pas son fonctionnement sur cet OS.

Nous sommes la 2eme équipe en charge de cette application, pour l'UE projet de L3 informatique.
Nous sommes encore en train d'apprendre à développer, puisqu'aucun de nous n'a d'expérience pro,
veuillez nous excusez si certain bout de code sont "moches".

Le nom du projet est "flutterapk" puisqu'au je faisais des test pour faire installer
l'application sur le téléphone de Julia (ce fût un enfer). Au final, c'est rester le main dossier
puisque j'ai trouvé les bonnes configuration pour que cet apk s'installe sans problème sur son
téléphone.
